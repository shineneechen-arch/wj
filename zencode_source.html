README.md

ZenCode Astrolabe

ZenCode 是一个基于 Three.js 的 3D 星盘渲染实验项目，用于把黄道坐标、赤道坐标、宫位系统和传统术数映射放进同一套可视化场景中。

Features

支持黄道/赤道坐标切换、行星轨道拖拽、整宫制与 Placidus 宫制切换、二十八宿叠加显示、基础相位线实时重算。

Local Build

npm install · npm run dev · npm run build
